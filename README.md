# Automatic Tool

- Dự án đa số chạy bằng pyautogui và có nhiều chức năng (gồm hợp pháp và phạm pháp)
- Môi trường hỗ trợ : Windows

---

## Cài đặt

- Clone repository:

```bash
git clone https://github.com/<username>/<ten-du-an>.git
```
-Run:
```bash
cd automatic_tool
python pyauto.py
```
## Yêu cầu:

-Python version : 3.12.x
