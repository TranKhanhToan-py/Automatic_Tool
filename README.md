## Installation

- Clone repository:

```bash
git clone https://github.com/<username>/<ten-du-an>.git
```
Vào thư mục vừa clone và chạy:
```bash
cd ten-du-an
bash run.sh
```


## Usage
```bash
python pyauto.py
```
